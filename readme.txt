== Installation ==

Manual installation:

1. Upload the 'BrokersWeb Ads' folder to the `/wp-content/plugins/` directory

Installation using 'Add New Plugin'

1. From your Admin UI (Dashboard), use the menu to select Plugins -> Add New
1. Search for 'BrokersWeb Ads'
1. Click the 'Install' button to open the plugin's repository listing

Activiation and Use

1. Activate the plugin through the 'Plugins' menu in WordPress
1. From your Admin UI (Dashboard), use the menu to select Options -> BrokersWeb Ads
1. Configure settings, and save.
